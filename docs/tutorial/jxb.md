# 教学班

教学班示例：`(2025-2026-2)-77101504-02`

使用精确的教学班查询能能减少教务系统返回的数据量，有利于加快抢课。

只有当使用精确教学班查询时，才适合直接使用 `Course.try_select_0` 方法

如果不了确定教学班，也可直接尝试使用课程号，示例：`77101504`

课程号会返回相同课程下的所有教学班，包括不同上课时间和教师等，建议手动调用 `Client.select_course(course_id, course_do_id)`

## 如何获得精确的教学班参数？

- 在抢课尚未开始的前一天，通常可以预览可选课程，这时候可以直接复制教学班为抢课做准备
- 留意官网选课公告，通常会将教学班在Execl中一同发出来

::: info
即使不使用`lnuElytra`,直接通过教务系统网页手工抢课，也可通过教学班搜索加快抢课速度
:::
